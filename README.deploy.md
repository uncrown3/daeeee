One-click Vercel deploy

Click the button below to import and deploy this project to Vercel in one step.

![Deploy to Vercel](https://vercel.com/button)

Import link (GitLab):

https://vercel.com/new/git/external?repository-url=https://gitlab.com/amir230/test.git

After import you will need to add environment variables listed in `.env.example` (AUTH_SECRET, APP_USERS_JSON, BLOB_READ_WRITE_TOKEN, CRON_SECRET).
